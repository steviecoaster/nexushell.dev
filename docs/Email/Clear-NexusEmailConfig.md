# Clear-NexusEmailConfig

## SYNOPSIS
Clears and disables the Smtp configuration in Nexus

## SYNTAX

```
Clear-NexusEmailConfig [-Force] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Clears and disables the Smtp configuration in Nexus

## EXAMPLES

### EXAMPLE 1
```
Clear-NexusEmailConfig
```

### EXAMPLE 2
```
Clear-NexusEmailConfig -Force
```
