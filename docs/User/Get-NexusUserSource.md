# Get-NexusUserSource

## SYNOPSIS
Retrieve a list of available user sources

## SYNTAX

```
Get-NexusUserSource [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Retrieve a list of available user sources.
These sources are things that can handle authentication to Nexus

## EXAMPLES

### EXAMPLE 1
```
Get-NexusUserSource
```
