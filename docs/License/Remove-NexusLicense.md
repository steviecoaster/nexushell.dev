# Remove-NexusLicense

## SYNOPSIS
Removes the license from a Nexus instance

## SYNTAX

```
Remove-NexusLicense [-Force] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Removes the license from a Nexus instance

## EXAMPLES

### EXAMPLE 1
```
Remove-NexusLicense
```

### EXAMPLE 2
```
Remove-NexusLicense -Force
```
