# Set-NexusAnonymousAuth

## SYNOPSIS
Turns Anonymous Authentication on or off in Nexus

## SYNTAX

```
Set-NexusAnonymousAuth [-Enabled] [-Disabled] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Turns Anonymous Authentication on or off in Nexus

## EXAMPLES

### EXAMPLE 1
